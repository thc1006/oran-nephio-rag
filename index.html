⚡ 快速開始

三步驟快速部署你的智能問答系統

1

克隆 & 安裝

git clone https://github.com/company/oran-nephio-rag.git
cd oran-nephio-rag
pip install -r requirements.txt
2

設定環境

cp .env.example .env
# 編輯 .env 並設定 ANTHROPIC_API_KEY
3

初始化 & 運行

python -c "from src import create_rag_system; rag = create_rag_system(); rag.build_vector_database()"
python -c "from src import quick_query; print(quick_query('什麼是 Nephio？'))"

🐳 或使用 Docker 一鍵部署

docker-compose -f docker-compose.prod.yml up -d
